About

Inspiration

Our mutual love for powerlifting.


What it does

Upon completing a quick survey, RoboTrainer will create a personalized workout program in the form of an excel sheet.


How we built it

Using python, the flask library coupled with some HTML & CSS, an SQL database and the sqlAlchemy python library.


Challenges we ran into

Teaching ourselves how to use the many tools required to build RoboTrainer, as well as how to connect them all together, all during the span of the hackathon.


Accomplishments that we're proud of

Having a working prototype for an application that can help improve people's health. Completing a project in our first hackathon.


What we learned

How to write HTML & CSS for the first time, how to research relevant python libraries according to our needs, how to navigate GitHub to collaborate with a team.


What's next for RoboTrainer

Delivery of a freestanding website/web application.
